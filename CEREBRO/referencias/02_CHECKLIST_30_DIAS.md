## SEMANA 1: FOUNDATION (Setup + Primeros Productos)

### 📱 GUMROAD - Crear Primer Producto (Día 1-3)

**Día 1: Setup + Planning**
- [ ] Crear cuenta Gumroad (5 min)
- [ ] Elegir nicho específico:
  - Opción 1: Prompts para [tu profesión]
  - Opción 2: Mini-curso sobre [tu expertise]
  - Opción 3: Templates/recursos para [problema específico]
- [ ] Investigar competencia en Gumroad ($0, 20 min)
- [ ] Decidir precio: $29, $49, o $99 (recomendado $49)
- **Deliverable:** Nicho + pitch decidido

**Día 2-3: Crear Producto**
- [ ] Generar contenido con ChatGPT (2-3 horas)
  - Prompt: "Create 50 professional prompts for [nicho] in a well-formatted PDF"
  - Output: Copiar a documento Word
- [ ] Diseñar cover en Canva (30 min)
  - Canva.com → Business Card size (se ve como thumbnail)
  - Colores bold, títulos grandes, legible
- [ ] Subir a Gumroad:
  - Crear producto new
  - Upload PDF
  - Upload cover image
  - Escribir descripción (copy de ChatGPT)
  - Set price: $49
  - Publish
- [ ] Crear simple landing text: "50 [nicho] Prompts - Ready to Use"
- **Deliverable:** Primer producto live en Gumroad

**Día 3 Afternoon: Marketing Setup**
- [ ] Crear post en Twitter: "Just launched 50 prompts for [nicho] on Gumroad. Used by [credibility]. Link in bio. $49"
- [ ] Post en ProductHunt trending (si aplica, free)
- [ ] Post en 2-3 subreddits relacionados
- **Goal:** 50-100 primeras visitas

**Tracking:** Visitas esperadas: 50-100 | Conversiones esperadas: 1-2 | Ingreso: $50-100

### 📝 MEDIUM - Write First 5 Articles (Día 3-5)

**Día 3: Setup Account**
- [ ] Create Medium account (gratuito)
- [ ] Complete profile:
  - Bio: "[Expertise] writer focusing on [nicho]"
  - Photo: profesional o avatar
  - Links: Twitter, Gumroad
- [ ] Aplicar a Medium Partner Program
  - Settings → Earnings → Apply (gratuito)
  - Puede tomar 1-2 semanas aprobación
- **Deliverable:** Account ready, Partner Program aplicación enviada

**Día 4-5: Write 5 Articles**
- [ ] Tema 1: "How I [resultado] using [método IA]" (tutorial)
- [ ] Tema 2: "[Problema] Solved: [solución específica]"
- [ ] Tema 3: "5 [cosa] That Changed [aspecto]"
- [ ] Tema 4: "Why Most People Fail At [cosa] (and how to avoid it)"
- [ ] Tema 5: "[Tool/Service] Review 2024: Worth It?"

**Para cada artículo (30-45 min):**
1. Título: Optimizado para clicks (números, preguntas, curiosidad)
2. First paragraph: Hook fuerte (problema + solución promise)
3. Body: Usar ChatGPT para generar draft (2-3k palabras)
4. Editar y personalizar (20 min)
5. Agregar tags: [2-3 tags relevantes]
6. Publish + compartir en Twitter

**Tracking:** Articles live: 5 | Views esperadas: 100-300 | Ingreso: $0 (esperando aprobación)

### 📰 SUBSTACK - Launch Newsletter (Día 1-2)

**Día 1: Setup**

  - Bio: Problema que resuelves + unique angle

**Día 2: Write + Send First Newsletter**
- [ ] Escribir primer email (30-45 min):

  - Intro: Problema personal o universal
  - Body: 3-5 insights principales
  - CTA: "Reply with your thoughts" o "Share this"
  - Firma personal
- [ ] Usar ChatGPT para draft:
  - Prompt: "Write a newsletter about [tema] for [audience]. Make it personal, not corporate. Include 3 insights and a question"
- [ ] Editar + personalizar (15 min)
- [ ] Send a: Friends + Twitter followers (manual invite)
- [ ] Post en Twitter: "Launching newsletter about [tema]. First issue just went out. Subscribe: [link]"

**Tracking:** Newsletter sent: 1 | Suscriptores esperados: 30-100 | Ingreso: $0
