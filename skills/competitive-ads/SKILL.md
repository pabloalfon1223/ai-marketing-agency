# SKILL: Competitive Ads Extractor

Analizás anuncios de la competencia para extraer los patrones que convierten.
Output: matriz de hooks, copy patterns, y creativos listos para adaptar.

## PROCESO

### PASO 1: Identificar competidores por proyecto

**MENTE PAUSADA** (bienestar mental):
- Calm, Headspace, Insight Timer (apps)
- Cursos de mindfulness en Hotmart/Gumroad
- Coaches de bienestar en Instagram
- Búsqueda: "ansiedad", "meditación", "ruido mental", "regulación emocional"

**POLT MOBILIER** (muebles a medida):
- Outras mueblerías a medida en Argentina (CABA/GBA)
- Catálogos de IKEA, Easy (para ver qué NO hacer)
- Carpinteros/diseñadores en Instagram con +10k seguidores
- Búsqueda: "muebles a medida", "placard a medida", "diseño de interiores"

**CEREBRO** (ideas/negocios digitales):
- Newsletters de negocios (Morning Brew, The Hustle en español)
- Cursos de negocios digitales
- Búsqueda: "ganar dinero online", "negocio digital", "emprendimiento"

---

### PASO 2: Framework de análisis (por cada anuncio)

Para cada anuncio competidor analizá:

```
ANUNCIO ANALIZADO:
URL/Descripción: [donde lo viste]
Marca: [nombre del competidor]

HOOK (primeras 3 segundos o primera línea):
→ Tipo: [pregunta / afirmación / dato / historia / provocación]
→ Emoción: [curiosidad / miedo / esperanza / enojo / humor]
→ Copy exacto: "[texto del hook]"

PROBLEMA QUE TOCA:
→ Pain point: [qué dolor nombra]
→ Audiencia implícita: [a quién le habla]

SOLUCIÓN PRESENTADA:
→ Cómo presenta el producto: [directo / indirecto / testimonial]
→ Diferencial destacado: [qué lo hace único]

CTA:
→ Acción pedida: [comprar / saber más / descargar / registrarse]
→ Urgencia: [hay o no hay urgencia]
→ Fricción: [alta / media / baja]

ESTIMACIÓN DE PERFORMANCE:
→ Engagement visible: [comentarios, compartidos]
→ ¿Llevan tiempo corriendo? [indicador de que funciona]
→ Score estimado: [1-10]
```

---

### PASO 3: Extraer patrones

Después de analizar 5-10 anuncios, identificá:

**HOOK PATTERNS QUE FUNCIONAN EN EL NICHO:**
```
Patrón 1: [descripción] — Ejemplo: "¿Por qué [X] aunque hagas [Y]?"
Patrón 2: [descripción] — Ejemplo: "La razón por la que [público] [problema]"
Patrón 3: [descripción] — Ejemplo: "[Número] de cada [X] personas tienen [problema]"
```

**EMOCIONES QUE CONVIERTEN:**
```
Principal: [emoción dominante en los mejores anuncios]
Secundaria: [emoción de apoyo]
A evitar: [qué emociones no funcionan en el nicho]
```

**FORMATO QUE DOMINA:**
```
Video: [qué tipo de video funciona más]
Imagen: [qué tipo de imagen]
Copy: [longitud, estructura]
```

---

### PASO 4: Generar 5 variantes propias

Basado en los patrones extraídos, creá 5 versiones de anuncio:

```
VARIANTE 1 — [Nombre descriptivo]
Hook: [primeras 3 segundos / primera línea]
Body: [desarrollo en 2-3 líneas]
CTA: [acción + beneficio]
Visual brief: [qué imagen/video va con este copy]
Audiencia Meta sugerida: [intereses, comportamientos, datos demográficos]
Budget test sugerido: $5-10 USD/día por 3 días
```
