# 📐 FORMATOS Y ESTRUCTURA DE CONTENIDO — MENTE PAUSADA

---

## FORMATO 1 — REEL

**Duración ideal:** 30–60 segundos (máximo 90)  
**Producción:** IA + texto en pantalla / grabación / mix  
**Prioridad:** Alta — mayor alcance y tracción orgánica

### Estructura base de un Reel de Mente Pausada

```
[0–3 seg]  HOOK VISUAL + TEXTO
           → Para el scroll. Primera frase en pantalla o narrada.
           → Sin música ruidosa, sin transición caótica.

[3–15 seg] DESARROLLO / GANCHO PROFUNDO
           → Amplía el dolor o la promesa del hook.
           → El usuario decide si sigue o se va. Tiene que ver que "vale la pena."

[15–45 seg] CONTENIDO CENTRAL
           → Micro-práctica / dato educativo / historia / herramienta
           → Ritmo pausado. Texto en pantalla. Voz suave si aplica.

[45–55 seg] CIERRE + CTA
           → Frase de cierre memorable.
           → Acción: "Guardá esto", "Comentá X", "Escribime PAUSA" (conversión)

[55–60 seg] SLIDE/FRAME FINAL (opcional)
           → Logo o frase de marca simple.
```

### Tipos de Reel que producimos

| Tipo | Descripción | Pilar |
|---|---|---|
| **Reel guiado** | El video lleva al usuario por una micro-práctica | 1 |
| **Reel educativo** | Explica algo sobre la mente de forma simple | 2 |
| **Reel relatable** | Historia o situación con la que el usuario se identifica | 3 |
| **Reel de reflexión** | Frase + b-roll + narración suave | 3/4 |
| **Reel de conversión** | Presenta el producto al final de forma natural | 5 |
| **Reel de tendencia** | Formato viral adaptado al tono de marca | 6 |

### Notas de producción IA

- Paleta: tierra, verde musgo, crema, beige cálido
- Sin efectos de transición rápidos
- Tipografía: grande, legible, estilo minimalista (no decorativa)
- Ritmo: lento-medio. Nunca frenético.
- Música: ambient, instrumental suave, lo-fi calmado. Sin drops ni beats agresivos.

## FORMATO 2 — CARRUSEL

**Cantidad de slides:** 5–10 (ideal 6–8)  
**Producción:** Diseño estático / IA generativa / Canva  
**Prioridad:** Alta — mejor para guardados y educación profunda

### Estructura base de un Carrusel

```
SLIDE 1 — HOOK
  → Promesa clara. Hace querer pasar el dedo.
  → Formato frecuente: número + tema ("5 señales de que...") o pregunta.

SLIDE 2 — CONTEXTO / EMPATÍA
  → Valida el dolor del usuario antes de dar contenido.
  → "Si alguna vez sentiste que..."

SLIDES 3-7 — CONTENIDO
  → Un punto por slide. Corto, claro, aplicable.
  → Cada slide puede ser leído en 5 segundos.

SLIDE PENÚLTIMO — RESUMEN / FRASE CLAVE
  → La conclusión o el insight más importante.
  → Lo que el usuario se lleva aunque no recuerde el resto.

SLIDE FINAL — CTA
  → Acción concreta: guardar, compartir, escribir "PAUSA", seguir.
  → Mención suave del producto si aplica.
```

### Tipos de Carrusel que producimos

| Tipo | Descripción | Pilar |
|---|---|---|
| **Lista educativa** | "X cosas que..." / "X señales de..." | 2 |
| **Paso a paso** | Guía de una práctica concreta | 1 |
| **Comparativo** | "Esto vs. esto" (ej: descanso real vs. distracción) | 2/4 |
| **Historia visual** | Narrativa slide a slide con frase final | 3 |
| **Mini-guía** | Carrusel que funciona como recurso de referencia | 1/4 |

### Notas de diseño

- Fondo: colores de la paleta de marca (no blanco puro)
- Tipografía: coherente con los Reels
- Slide 1: impacto visual fuerte
- Slides interiores: limpio, con espacio, sin saturar
- Último slide: siempre con handle (@mentepausada) y CTA

## FORMATO 3 — B-ROLL + FRASE / POST VISUAL

**Uso:** Contenido de alcance, frases de marca, respiración visual  
**Producción:** B-Roll con texto superpuesto / imagen estática / diseño mínimo  
**Prioridad:** Media — complementa el calendario y refuerza identidad visual

### Estructura

```
VISUAL: B-Roll o imagen evocadora (naturaleza, manos, texturas, luz)
FRASE: Texto superpuesto — corto, poético, real
AUDIO: Música ambiente suave (sin narración)
```

### Tipos de B-Roll que producimos

| Tipo | Ejemplo de frase | Pilar |
|---|---|---|
| **Frase de marca** | "Tu mente no está rota. Está cansada." | 3 |
| **Permiso** | "Hoy podés hacer una sola cosa. Está bien." | 4 |
| **Recordatorio de práctica** | "Tres respiraciones. Ahora." | 1 |
| **Reflexión breve** | "El descanso también es trabajo." | 4 |
| **Cita externa** | Cita atribuida + fuente + frase de cierre | 2/6 |

## CALENDARIO SEMANAL SUGERIDO

| Día | Formato | Pilar |
|---|---|---|
| Lunes | Reel educativo o relatable | 2 / 3 |
| Martes | Carrusel (lista o paso a paso) | 1 / 2 |
| Miércoles | B-Roll + frase | 3 / 4 |
| Jueves | Reel guiado (micro-práctica) | 1 |
| Viernes | Reel de conversión o entretenimiento | 5 / 6 |
| Sábado | B-Roll o post visual tranquilo | 3 / 6 |
| Domingo | Opcional / stories de conexión | 3 |

**Ritmo mínimo:** 4 piezas por semana (2 Reels + 1 Carrusel + 1 B-Roll)  
**Ritmo ideal:** 6–7 piezas por semana

## ESTRUCTURA DE CAPTION (para todos los formatos)

```
LÍNEA 1: Reescritura del hook (igual o distinto al visual)
          → Tiene que funcionar solo, sin ver el video/carrusel

ESPACIO EN BLANCO

DESARROLLO: 2-4 líneas que amplían el tema
             → Sin emojis en exceso. Tono de conversación.

ESPACIO EN BLANCO

CTA: Acción específica
     → "Guardalo para cuando lo necesites"
     → "¿Te pasó esto? Contame en los comentarios"
     → "Escribime PAUSA y te mando el recurso"

HASHTAGS: 5–10 relevantes (no saturar)
```
