## Agentes del sistema

### CAPA 1 — INTELIGENCIA DE MERCADO

**SCOUT** — Buscar ideas de negocio digital con tracción real
- Criterios: remoto, USD, recurrente, faceless, ejecutable solo
- Output: tabla de 5 ideas con score (mercado / competencia / factibilidad / recurrente)

**COMPETE** — Análisis competitivo profundo
- Input: idea seleccionada
- Output: mapa de competidores + gap explotable + ángulo diferenciador

**AUDIENCE** — Definir el cliente ideal con precisión
- Input: idea + ángulo diferenciador
- Output: 3 avatares + ICP seleccionado + mensaje central + palabras exactas del ICP

**TREND** — Detectar tendencias emergentes antes de que sean obvias
- Input: nicho
- Output: 3 tendencias rankeadas por urgencia de acción

### CAPA 2 — PRODUCTO

**PRODUCT** — Diseñar el stack completo de 4 tiers
- Input: idea validada + ICP
- Output: documento con entry/core/sub/HT definidos (nombre, promesa, formato, contenido mínimo)

**CREATOR** — Crear el contenido real del producto
- Formatos: guiones para audio, cursos en video/slides, PDFs, templates, comunidades
- Output: contenido listo para producir sección por sección

**PRICING** — Definir estructura de precios y LTV
- Input: stack de productos
- Output: tabla de precios + proyección de LTV por cliente + estrategia de descuento

### CAPA 3 — MARCA

**BRAND** — Crear identidad de marca completa (faceless)
- Output: nombre (5 opciones), tagline (3 opciones), voz/tono, paleta hex, tipografía, brief visual AI

**COPY** — Escribir copy que vende sin sonar a vendedor
- Frameworks: PAS, BAB, AIDA, 4U
- Tipos: headlines, bullets, descripciones, CTAs, bios, hooks, scripts faceless

**LANDING** — Arquitectura y copy completo de landing pages
- Estructura: 12 secciones (hero → problema → solución → cómo funciona → precio → FAQs → CTA)
- Output: documento sección por sección listo para Lovable/Carrd

**SEO** — Investigación de keywords y optimización
- Output: keyword principal + 10 relacionadas + estructura de contenido + meta tags + YouTube SEO

### CAPA 4 — MARKETING

**CONTENT** — Estrategia y piezas de contenido faceless
- Formatos: Reels con texto+voz AI, carruseles, screencast, podcasts, newsletters, threads X
- Output: pieza completa con hook + cuerpo + CTA + brief visual + prompt AI + script ElevenLabs

**EMAIL** — Secuencias de email automatizadas
- Tipos: bienvenida (5 emails) / lanzamiento (7 emails) / nurture / post-compra / win-back
- Output: asunto + preview text + cuerpo + CTA + tag de segmentación

**ADS** — Creativos de publicidad faceless
- Formatos: Meta imagen/video, YouTube pre-roll, X Ads
- Output: 3 variantes de copy + prompt visual + audiencia Meta + presupuesto mínimo de test

**FUNNEL** — Arquitectura del embudo completo
- Mapa: tráfico frío → contenido → lead magnet → bienvenida → entry → upsell → core → sub → HT
- Output: diagrama completo + métricas de conversión target por etapa

### CAPA 5 — VENTAS

**SALES** — Sales page y VSL (Video Sales Letter) faceless
- Estructura: 15 secciones (headline → opener → agitación → producto → precio → garantía → FAQs → cierre)
- VSL: script completo para ElevenLabs + instrucción de pantalla por segmento

**CHECKOUT** — Optimizar el proceso de pago
- Elementos: order bump (USD 17-37) + OTO post-compra + downsell + thank you page
- Output: configuración completa + copy de cada elemento

**UPSELL** — Arquitectura del backend y ascenso de clientes
- Secuencia: post-entry → post-core → invitación HT (días 7/14/21/30)
- Output: secuencia de emails + timing + lógica de oferta por tier

### CAPA 6 — OPERACIONES

**AUTO** — Diseñar y construir automatizaciones en N8N
- Automatizaciones prioritarias:
  1. Nuevo suscriptor → secuencia bienvenida
  2. Nueva compra → onboarding automático + Notion + comunidad
  3. Content pipeline semanal (Claude API → borrador → aprobación → visual)
  4. Scout de tendencias semanal → Notion
  5. Reporte KPIs dominical → WhatsApp
  6. Cancelación suscripción → win-back
- Output: diagrama ASCII + nodos N8N + código de transformación + instrucciones de test

**ANALYTICS** — Analizar métricas y recomendar experimentos
- Métricas core: opt-in rate (>30%), conversión entry (>3%), conversión core (>20%), churn (<5%)
- Output: diagnóstico + causa raíz + 1-3 experimentos concretos + forma de medir

**FINANCE** — Trackear ingresos, gastos y proyecciones
- Reporte mensual: MRR actual/anterior, proyección 3-6 meses, breakdown por producto, margen neto
- Output: análisis + palanca principal a mover para crecer más rápido
