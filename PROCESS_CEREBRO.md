# Cerebro - Flujo de Ideación y Validación $20k+

## 📋 Resumen Ejecutivo

Cerebro es un **sistema de generación y validación de ideas de negocio** escalables a $20k-50k/mes.

- **Objetivo:** Encontrar oportunidades reales de ingresos altos
- **Metodología:** Validate small before scaling big ($500/mo → $5k → $20k)
- **Input:** Análisis de tendencias + demanda real
- **Output:** Ideas ranqueadas por viabilidad + tracking de progreso
- **Horizonte:** Exploración continua, MVP testeable en 4-8 semanas

### Fase 2: Validación de Mercado

**Skill:** `validador-mercado-cerebro`

**Para cada idea con score >70:**

```
1. DEMANDA (40%)
   ├─ Google Trends: búsquedas, crecimiento
   ├─ SEO difficulty + search volume
   ├─ Reddit/Quora: preguntas reales de gente
   ├─ ProductHunt: existe? Qué hace?
   └─ Resultado: Demand score 0-100

2. ESCALABILIDAD (40%)
   ├─ Modelo de ingresos: one-time, recurrente, hybrid
   ├─ Precio potencial: mín-máx realista
   ├─ Proyección usuarios/clientes a $20k
   ├─ Posibilidad de automatización
   └─ Resultado: Scalability score 0-100

3. COMPETENCIA (20%)
   ├─ Competidores existentes: cuántos, qué hacen
   ├─ Gaps en mercado: qué no ofrece nadie
   ├─ Barriers to entry: capital, expertise, tiempo
   ├─ Diferenciador tuyo: qué es único
   └─ Resultado: Competition score 0-100

OVERALL SCORE = (Demand × 0.4) + (Scalability × 0.4) + (Competition × 0.2)
```

**Ejemplo de Validación: AI Content SaaS**

```
DEMAND VALIDATION
├─ Google Trends: "AI content writing" 45k búsquedas/mes, +45% YoY ✓
├─ Reddit r/freelancers: 50+ posts/semana pidiendo herramienta ✓
├─ ProductHunt: 3 lanzamientos similares en 90 días ✓
├─ Competidores: Copy.ai ($49/mo), Jasper ($99/mo), BUT gaps en [X] ✓
└─ Score: 85/100

SCALABILIDAD
├─ Modelo: SaaS recurrente (MRR) ✓✓
├─ Precio realista: $29-99/mes según features ✓
├─ Path a $20k: 200+ suscriptores a $100/mo promedio ✓
├─ Automatización: 100% automática después launch ✓
└─ Score: 90/100

COMPETENCIA
├─ Existentes: 15+ herramientas similares (saturado) ✗
├─ Gaps: Falta especialización en [nicho específico] ✓
├─ Barriers: $5k-10k en desarrollo, 8 semanas ✓
├─ Diferenciador: Tuyo es 3x más barato OR específico para [target] ✓
└─ Score: 60/100

OVERALL: (85×0.4) + (90×0.4) + (60×0.2) = 79/100
```

**Decision Rule:**
```
Score >75: PURSUE - crear MVP
Score 60-75: RESEARCH - validar más antes
Score <60: BACKLOG - esperar, o pivot diferente
```

### Fase 3: MVP & Progressive Scaling

**Ciclo: Validate Small First**

```
STEP 1: Pre-launch validation (1-2 semanas)
├─ Landing page simple con descripción
├─ Recopilar 100+ emails interesados
├─ Google Form: "¿Pagarías $X por esto?"
└─ Meta: 30%+ diciendo sí

STEP 2: MVP Launch (4-6 semanas)
├─ Crear versión mínima funcional
├─ Lanzar a 50-100 early adopters
├─ Recibir feedback intenso
├─ Iterate rápido (3-4 cycles)
└─ Meta: 20%+ satisfacción (NPS >30)

STEP 3: Monetización PEQUEÑA (2-4 semanas)
├─ Cobrar desde día 1 (aunque sea barato)
├─ Precio bajo: $9-29/mes o one-time $99
├─ Meta: $500/mes en ingresos
└─ Proof of concept

STEP 4: Scale a $5k/mes (4-8 semanas)
├─ Mejorar producto basado en feedback
├─ Aumentar marketing (ads, content, partnerships)
├─ Optimize pricing
└─ Meta: $5,000/mes MRR

STEP 5: Path to $20k (12-16 semanas total)
├─ Automatizar operaciones
├─ Crear system/process docs
├─ Potentially hire freelancer si needed
└─ Meta: $20,000/mes MRR
```

**Ejemplo Timeline AI Content SaaS:**
```
Week 1-2: Landing page + signup list (1k emails)
Week 3-6: Desarrollar MVP (web app básica)
Week 7: Lanzar con 50 early adopters + cobrar $19/mo
Week 8-10: Feedback + iteraciones rápidas
Week 11-14: Escalar a $5k/mes (250+ subscribers)
Week 15-20: Marketing + optimize a $20k/mes (1000+ subscribers)
```

### Fase 4: Monthly Retrospective

**Tarea:** `cerebro-monthly-retrospective` (1° de mes, 9 AM)

**Qué hacer:**

```
1. ANALYZEPROJECTS
   ├─ Ideas que llegaron a $500+: keep, amplify
   ├─ Ideas que bajaron en demanda: abandon o pivot
   ├─ Ideas con MVP lanzado: qué funcionó, qué no
   └─ Ideas aún en validación: progreso?

2. MAKE DECISIONS
   ├─ Enfoque mes próximo: Top 3 ideas a acelerar
   ├─ Abandon: cuáles ideas dejar (por ahora)
   ├─ Double-down: cuáles escalan más
   ├─ Pivot: ideas que necesitan cambio de dirección
   └─ New: ideas nuevas a considerar

3. DOCUMENT LEARNINGS
   ├─ Qué funcionó: "SaaS con especialización gana"
   ├─ Qué fracasó: "Oversaturated markets son difíciles"
   ├─ Pattern: "Ideas trending en TikTok = demanda real"
   ├─ Next: "Enfatizar validación antes de coding"
   └─ Compartir estos learnings con Mente Pausada + Polt

4. OUTPUT
   ├─ Archivo: /output/cerebro-retrospective-{MONTH}.md
   ├─ Incluye: Decisiones, datos, reasoning
   └─ Próximos pasos claramente definidos
```

## 🛠️ Integración Técnica

### Base de Datos (Schema)

```sql
CREATE TABLE ideas (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  
  -- Descripción
  title VARCHAR(255) NOT NULL,
  description TEXT NOT NULL,
  idea_type VARCHAR(50) NOT NULL,  -- saas/servicios/ecommerce/creator/hybrid
  
  -- Validación (Scores)
  demand_score INTEGER,  -- 0-100
  scalability_score INTEGER,  -- 0-100
  competition_score INTEGER,  -- 0-100
  overall_score INTEGER,  -- 0-100 (weighted)
  
  -- Evidencia
  demand_evidence TEXT,  -- URLs, keywords, búsquedas
  competitor_evidence TEXT,  -- qué existe
  gaps_identified TEXT,  -- oportunidades
  
  -- MVP Plan
  mvp_outline TEXT,  -- descripción MVP
  estimated_development_time INTEGER,  -- días
  estimated_capital_needed DECIMAL(10,2),
  
  -- Lifecycle
  status VARCHAR(30),  -- idea_cruda/validada/mvp_building/mvp_launched/scaling_to_5k/scaling_to_20k/exitosa/pivotada/descartada
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  validated_at DATETIME,
  launch_date DATETIME,
  
  -- Revenue Tracking
  current_monthly_revenue DECIMAL(10,2) DEFAULT 0,
  reached_500 BOOLEAN DEFAULT false,
  reached_5k BOOLEAN DEFAULT false,
  reached_20k BOOLEAN DEFAULT false,
  
  -- Learnings
  learnings TEXT,
  pivot_reason VARCHAR(255),
  abandon_reason VARCHAR(255)
);

CREATE INDEX idx_overall_score ON ideas(overall_score);
CREATE INDEX idx_status ON ideas(status);
```

### API Endpoints

**Generate Ideas**
```
POST /api/ideas/generate
Input: {
  available_capital: 5000,
  available_time: "part_time|full_time",
  skills: ["marketing", "coding", "writing"]
}
Output: 20+ ideas with descriptions and initial estimates
```

**Validate Idea**
```
PUT /api/ideas/{id}/validate
Skill: validador-mercado-cerebro
Output: {
  demand_score: 85,
  scalability_score: 90,
  competition_score: 60,

```

Input: {
  status: "mvp_launched",
  current_revenue: 500,
  learnings: "Early adopters love the personalization feature"
}
```

**Get Trending Ideas**
```
